# Karakeep Social Button - Vibe Coding Project

A browser extension that allows users to seamlessly save links, images, and posts directly from social media platforms to their Karakeep workspace.

## Features

- **Direct Integration:** Injects a custom Karakeep save button into the native interface of supported platforms.
- **Supported Platforms:** Currently works on Instagram, Pinterest and Twitter (X).
- **Easy Saving:** Click the Karakeep button to open a popup where you can add a title and tags before saving directly to the Karakeep API.

## Installation

1. Clone this repository.
2. Search your Karakeep API and add it to the request.js file.
3. Open your browser's extension management page (e.g., `chrome://extensions/`).
4. Enable "Developer mode".
5. Click "Load unpacked" and select the directory containing this repository.

## Usage

Once installed, browse to a supported social media platform. You will notice the Karakeep save button integrated alongside the native sharing/saving options. Click it to bring up the save dialog, fill in any relevant details, and save your content directly to Karakeep.
